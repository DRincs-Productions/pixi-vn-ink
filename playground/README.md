# Vite Plugin Test

Progetto minimo per testare i plugin Vite:

- `vitePluginPixivn()`
- `vitePluginInk({...})`

## Avvio

Dalla root di questo repository:

```bash
cd playground
npm install
npm run dev
```

## Build

```bash
npm run build
```

Dopo la build/dev, verifica:

- `public/ink-json/**/*.gen.json`
- `src/assets/ink-manifest.gen.json`
